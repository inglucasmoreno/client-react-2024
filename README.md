# Template - FrontEnd - React 2024

## Tecnologias utilizadas
- Vite
- React
- Shadcn/ui
- TailwindCSS

## Preparacion del proyecto

- Instalacion de dependencias
```
pnpm install
```

- Ejecutar servidor de desarrollo
```
pnpm run dev
```

- Generacion de archivo de produccion
```
pnpm run build
```
